interface IInAppMessageWebViewClientListener
Content copied to clipboard

The IInAppMessageWebViewClientListener is called at specific events during the display of an Html In-App Message. Button clicks that occur inside an HTML In-App Message are routed to this listener and not the IInAppMessageViewLifecycleListener. However, the display lifecycle of the HTML In-App Message is still handled by the IInAppMessageViewLifecycleListener. See InAppMessageWebViewClient.
