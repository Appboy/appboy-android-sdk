Returns the channel id for a valid NotificationChannel, creating one if necessary.

  1. First, if Constants.APPBOY_PUSH_NOTIFICATION_CHANNEL_ID_KEY key is present in {@literal notificationExtras}'s and is the id of a valid NotificationChannel, this id will be returned.
  2. Next, if the channel with id Constants.APPBOY_PUSH_DEFAULT_NOTIFICATION_CHANNEL_ID exists, then Constants.APPBOY_PUSH_DEFAULT_NOTIFICATION_CHANNEL_ID will be returned.
  3. Finally, if neither of the cases above is true, a channel with id Constants.APPBOY_PUSH_DEFAULT_NOTIFICATION_CHANNEL_ID will be created and Constants.APPBOY_PUSH_DEFAULT_NOTIFICATION_CHANNEL_ID will be returned.
