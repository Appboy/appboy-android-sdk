BrazeFeedActivity

open class BrazeFeedActivity : BrazeBaseFragmentActivity

The BrazeFeedActivity in an Activity class that displays the Braze News Feed Fragment. This class can be used to integrate the Braze News Feed as an Activity. Note: To integrate the Braze News Feed as a Fragment instead of an Activity, use the BrazeFeedFragment class.
