open fun requestContentCardsRefresh(fromCache: Boolean)
Content copied to clipboard

Requests an immediate refresh of Content Cards. If fromCache is true, data will be pulled from offline storage. If fromCache is false, the SDK will request an immediate refresh of Content Cards from the Appboy server. Results of both cached and non-cached refresh calls are reported asynchronously via events. Events can be subscribed to using subscribeToContentCardsUpdates If the network call returns before any subscribers are active, the result will be lost.

Parameters

fromCache

whether to request data from offline storage.
