Creates a View.OnClickListener that calls IInAppMessageViewLifecycleListener.onClicked.

IInAppMessageViewLifecycleListener.onClicked is called and can be used to turn off the close animation. Full and modal in-app messages can only be clicked directly when they do not contain buttons. Slideup in-app messages are always clickable.
